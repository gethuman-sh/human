# Amplitude Analytics

Query your Amplitude product analytics straight from `human`, so an agent can ground decisions in real user behavior. Explore events, run funnels and retention, and look up individual users.

- List tracked events with active user counts
- Analyze event trends over a date range
- Browse event and user property schemas
- Run funnel conversion across a sequence
- Measure retention day by day
- Search users by name, email, or ID
- View a user's full event history
- List saved behavioral cohorts
