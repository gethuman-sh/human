# Notion Knowledge Base

Pull pages and database content out of your Notion workspace from `human`, so an agent can read product docs, specs, and notes as it works. Search, fetch full pages as markdown, and browse database rows.

- Search the workspace for pages and databases
- Fetch full page content as clean markdown
- Read nested blocks, child pages, and tables
- Query database rows as structured data
- List every database shared with the integration
- View row properties with types resolved automatically
- Output as JSON or readable terminal tables
- Switch between multiple configured Notion accounts
