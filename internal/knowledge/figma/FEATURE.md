# Figma Designs

Reach into your Figma design files straight from `human`, so an agent can read what designers built without leaving the terminal. Inspect screens, pull component details, read feedback, and export images for downstream work.

- Browse pages and metadata inside a design file
- Inspect element size, text, type, and color
- List published components and their descriptions
- Read design comments with author and status
- Export design elements as PNG, JPG, SVG, or PDF
- List projects and files across a team
- Switch between multiple configured Figma accounts
