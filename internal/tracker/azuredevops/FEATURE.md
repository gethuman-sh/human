# Azure DevOps Tracker

Lets `human` work with work items in your Azure DevOps projects, so AI and human developers can track engineering work from the CLI.

- List open or all work items in a project
- Read a single work item with full details
- Create work items and parent-child links
- Edit work item title and description
- Add and read work item comments
- Move work items through available states
- Assign work items to team members
- Close work items when finished
