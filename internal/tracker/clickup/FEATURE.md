# ClickUp Tracker

Lets `human` manage tasks in your ClickUp workspace, so AI and human developers can track work and browse the workspace hierarchy from the CLI.

- List open or all tasks in a list
- Read a task by ID or custom ID
- Create tasks, subtasks, and comments
- Edit task title and description
- Move tasks through their statuses
- Browse spaces, folders, lists, and members
- Read and set task custom fields
- Assign tasks to workspace members
