# Jira Tracker

Lets `human` work with issues in your Jira projects, so AI and human developers can read requirements and track engineering work without leaving the CLI.

- List open or all issues in a project
- Read a single issue with full details
- Create new issues and subtasks
- Edit issue title and description
- Add and read issue comments
- Move issues through workflow statuses
- Assign issues to teammates
- Delete issues you no longer need
