# Shortcut Tracker

Lets `human` work with stories in your Shortcut workspace, so AI and human developers can read product requirements and track work from the CLI.

- List open or all stories in a group
- Read a single story with full details
- Create stories, bugs, chores, and subtasks
- Edit story title and description
- Add and read story comments
- Move stories through workflow statuses
- Assign stories to team members
- Delete stories you no longer need
