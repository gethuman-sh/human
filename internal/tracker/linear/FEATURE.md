# Linear Tracker

Lets `human` manage engineering work in your Linear teams, so AI and human developers can plan, track, and update tickets straight from the CLI.

- List open or all issues per team
- Read a single issue with full details
- Create new issues and subtasks
- Edit issue title and description
- Add and read issue comments
- Move issues through workflow states
- Assign issues to teammates
- Delete issues you no longer need
