# Slack Messaging

Send messages to a Slack channel and read its recent history from `human`, so an agent can post updates and catch up on conversation without opening Slack. Each configured channel is ready to use with a single command.

- Post a text message to a channel
- List the most recent channel messages
- Cap how many messages you fetch
- View history as a readable table
- Export message history as JSON
- Switch between multiple configured Slack channels
- Resolve bot tokens from env or vault
