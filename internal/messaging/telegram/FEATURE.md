# Telegram Inbox

Receive, inspect, and clear messages from a Telegram bot through `human`, so an agent can take instructions over Telegram safely. A default-deny allowlist keeps only approved people and chats visible.

- List pending messages from your bot
- Read full details of a single message
- Acknowledge messages to clear the queue
- View messages as a table or JSON
- Allowlist trusted users by default-deny
- Require explicit opt-in for group chats
- Warn when the allowlist is empty
- Switch between multiple configured Telegram bots
