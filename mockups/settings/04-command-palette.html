Command palette — search is the only navigation

The gear (or Ctrl+,) opens a Raycast-style palette. Every leaf of .humanconfig is indexed as a dotted path; typing fuzzy-matches across all sections at once and shows current values inline. Enter opens a tiny editor for the selected key — edit, ↵ to save, done. There is no section browsing at all.

+ fastest path to any single setting, fully keyboard-driven + scales to any config size — 6 sections or 60 + current values visible in the result list itself − no browsing: you must know (roughly) what you're looking for − multi-field flows (add a whole tracker) fit badly in a palette
